# Execute in DEV

1. Clone repository
2. Execute
```
npm install
```
3. Install Nest CLI
```
npm i -g @nestjs/cli
```

4. Up Database
```
docker-compose up -d
```

#Stack Used
* MongoDB 5
* Nest
